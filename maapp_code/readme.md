[PyQt5打包程序为exe可执行文件](https://www.jianshu.com/p/769ab3aa9d96)

    第一步:输入打包命令: 
    pip install pyinstaller
    cd D:/app/
    pyinstaller -Fw app.py      # window.py为程序的执行文件名
    pyinstaller -F -c app.py      # window.py为程序的执行文件名
    
    -F: 表示生成单个可执行文件
    
    -w: 表示去掉控制台窗口
    
    -c: 表示启动控制台
